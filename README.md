# Simpsons API

Aplicación Android desarrollada en Kotlin que consume datos de personajes de Los Simpson. El proyecto sigue una arquitectura modular separando las capas de presentación, dominio y datos para facilitar la escalabilidad y el mantenimiento.

La aplicación permite listar personajes obtenidos de un servidor remoto y ver sus detalles manteniendo una navegación fluida. Se utilizan componentes modernos de Android, inyección de dependencias y gestión de estados para asegurar un funcionamiento robusto.
